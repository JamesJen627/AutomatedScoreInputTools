# 00 原始需求（DEMAND）

本文档为 ASIT 产品的原始需求摘要；详细规格见 PRD 分册（01–13）。**V1.1 已与当前实现对齐。**

---

## 1. 平台

Windows 桌面软件（Electron）；离线优先；Excel 驱动。

---

## 2. 导入与导出

### 2.1 输入

用户导入成绩 Excel。计算前必须包含以下**逻辑字段**（表头见 §2.3）：

班级、考号、学号、姓名、性别、五项**成绩**、五项**得分占比**。

- 输入文件中**允许**存在中间「得分」列（如 `坐位体前屈得分`），导入时**按表头名称映射**，忽略得分列，不参与校验。
- 输入文件中**允许**使用**简写表头**（见 PRD §5.21 别名表），系统自动映射为规范表头。
- **单项成绩允许为空**（缺考/未测）：校验通过，计算时该项得 **0 分**，不计入该项贡献分。

### 2.2 输出

导出表为 **22 列**，在规范 21 列基础上末尾追加 **等级** 列。列顺序固定（成绩 / 得分 / 占比交错）：

| # | 表头 |
|---|------|
| 1–5 | 班级、考号、学号、姓名、性别 |
| 6–8 | 坐位体前屈成绩（单位：厘米）、坐位体前屈得分、坐位体前屈得分占比 |
| 9–11 | 800m成绩（单位：分·秒）、800m得分、800m得分占比 |
| 12–14 | 50m成绩（单位：秒）、50m得分、50m得分占比 |
| 15–17 | 立定跳远成绩（单位：米）、立定跳远得分、立定跳远得分占比 |
| 18–20 | 仰卧起坐成绩、仰卧起坐得分、仰卧起坐得分占比 |
| 21 | 总成绩 |
| 22 | 等级（优≥90%/良≥80%/合≥60%） |

说明：

- 「仰卧起坐成绩」**导出列名不含**「（单位：次）」；数值仍来自输入列「仰卧起坐成绩（单位：次）」。
- **等级**由系统根据该生**最高可获总成绩**（= Σ 100 × 各项目占比）的比例自动判定，不可手工编辑。
- 默认导出文件名：`原文件名_已计算.xlsx`。

### 2.3 规范表头（系统内部 / 编辑器默认表头）

```
班级、考号、学号、姓名、性别、
坐位体前屈成绩（单位：厘米）、坐位体前屈得分占比、
800m成绩（单位：分·秒）、800m得分占比、
50m成绩（单位：秒）、50m得分占比、
立定跳远成绩（单位：米）、立定跳远得分占比、
仰卧起坐成绩（单位：次）、仰卧起坐得分占比
```

---

## 3. 评分标准

- 以 **`docs/成绩单项评分表.xlsx`**（官方表）为权威来源；通过 `npm run import-rules` 生成 `ScoreRules/2025/rule.xlsx`。
- 评分规则按 **项目 × 性别 × 年级（初一/初二/初三）** 三维匹配；年级由**班级名称**推断（如 `701` → 初一，首位数字 7/8/9 对应初一/初二/初三）。
- **Floor Rule**：成绩落在两档之间时，取**下限**对应得分。
- **单位换算**：评分表单位与学生输入不同时先换算再查表（如立定跳远：输入米 → 查表厘米；800m/耐力跑：输入分·秒 → 内部秒）。
- 单项得分必须为官方表规定的 **20 档整数**（100, 95, 90, …, 10）；总成绩保留两位小数。
- 800m 在规则与别名中亦称为 **耐力跑**。

---

## 4. 计算前检查

必须具备 §2.3 全部逻辑字段（可通过别名满足）。缺少表头 → 提示用户修改（支持 Excel 在线编辑与保存）。

**必填非空**：班级、考号、学号、姓名、性别、全部占比。

**成绩列**：允许为空（缺考）；非空时须符合格式（见 PRD §2.4）。

检查通过后方可进入计算。

---

## 5. 计算、审核与导出

1. 用户确认后执行计算；展示各项目得分、总成绩及**计算轨迹**。
2. **审核**：对同一数据集执行两次独立计算，结果必须完全一致；否则禁止导出。
3. 存在**计算失败**（如缺少对应年级/性别评分标准）的学生时，禁止导出。
4. 导出前用户可在线编辑；导出结果可再次导入系统继续计算（等级列与中间得分列在导入时忽略）。

---

## 6. 成绩格式摘要

| 项目 | 输入格式 | 示例 |
|------|----------|------|
| 坐位体前屈 | 数字（厘米，可负） | 19、-2.1 |
| 800m / 耐力跑 | mm'ss" | 3'30" |
| 50m | 小数秒 **或** 计时器写法 | 7.56、**8"7**（= 8.7 秒） |
| 立定跳远 | 米 | 1.71 |
| 仰卧起坐 | 整数（次） | 58 |

---

## 7. 等级判定

以该生五项均为满分 100 时可获得的 **maxPossibleTotalScore** 为满分基准：

| 等级 | 条件 |
|------|------|
| 优 | 总成绩 ≥ 满分的 90% |
| 良 | ≥ 80% 且 < 90% |
| 合 | ≥ 60% 且 < 80% |
| 不合格 | < 60% |
